# MVP Payout Model

Use a net-margin split first, not a gross-sales split.

## Formula

```text
Sale price
- payment processing fee
- POD production cost
- shipping cost, if absorbed
- refund/misprint reserve
= net margin

Net margin × creator share = creator payout
Net margin × hub share = operator revenue
```

## Example

```text
$32 shirt sale
- $1.23 payment processing
- $13.00 POD shirt cost
- $5.00 shipping if included
= $12.77 net margin

Creator receives 70% of net: $8.94
Hub keeps 30% of net: $3.83
```

## Recommended early split

Use 70/30 net margin for early partner creators if the goal is recruiting trust.

Define “net” very clearly in the creator agreement.
